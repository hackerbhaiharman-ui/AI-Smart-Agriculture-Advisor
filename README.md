# AI-Powered Smart Agriculture Advisor

This project uses Artificial Intelligence to assist farmers in making informed decisions related to crop selection, irrigation, and fertilizer usage.

## Features
- Crop recommendation based on soil and weather
- Smart irrigation guidance
- Pest and disease risk alerts
- Farmer-friendly interface

## Technologies Used
- Python
- Machine Learning
- Flask / Streamlit
- Weather API

## Future Scope
- Mobile app integration
- IoT-based soil sensors
- Regional language support
